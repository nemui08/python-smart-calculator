# Python Smart Calculator

A CLI scientific calculator built with Python.

## Features

- Basic math operations (+ - * /)
- Advanced math functions
  - sqrt
  - sin
  - cos
  - tan
  - log
- Supports mathematical expressions
- History system
- 'ans' memory (use previous result)
- Error handling

## Project Structure

```
python-smart-calculator
│
├── calculator.py
└── README.md
```

## Example Usage

```python
>>> 2+3*5
17

>>> sqrt(16)+cos(0)
5.0

>>> sin(pi/2)
1.0

>>> ans*10
10

```

## How to run

Make sure Python is installed.

```
python calculator.py
```
